# Overfit Commercial Licensing & Support

Overfit is dual-licensed to meet the needs of both open-source developers and enterprise-grade commercial projects.

## ⚖️ Why a Commercial License?
[cite_start]The standard open-source version of Overfit is licensed under **GNU AGPLv3**[cite: 14]. This license requires that any software using Overfit (including via a network/SaaS) must also be open-sourced under the same license.

**The Commercial License is designed for businesses that:**
* Want to keep their proprietary source code private.
* Require guaranteed performance SLAs for mission-critical systems.
* Need professional support and custom hardware optimization (AVX-512, ARM Neon).
* Cannot comply with the reciprocal sharing requirements of the AGPLv3.

## 🚀 Enterprise Features
A commercial license grants access to:
1.  **Proprietary Use Rights:** Legal permission to integrate Overfit into closed-source commercial products.
2.  [cite_start]**Zero-Allocation Guarantees:** Consultation on implementing "Beast Mode" architectures for zero-GC impact in high-throughput environments.
3.  [cite_start]**Thread-Safe Parallel Inference:** Specialized patterns for multi-threaded scaling on high-core-count CPUs like Ryzen 9950X3D.
4.  **Custom Kernel Development:** Support for specialized neural network layers optimized for specific hardware.
